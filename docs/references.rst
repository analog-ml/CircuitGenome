References
==========

The design of CircuitGenome draws on the following publications:

1. **A Data-Driven Analog Circuit Synthesizer with Automatic Topology
   Selection and Sizing** — inspired the performance-dataset generation
   workflow using enumerated topology variants.

2. **FUBOCO: Structure Synthesis of Basic Op-Amps by FUnctional BlOck
   COmposition** — the primary reference for functional block composition,
   canonical module interfaces, and topology templates.

3. **A Functional Block Decomposition Method for Automatic Op-Amp Design** —
   used for the functional block recognition rules and the hierarchical
   decomposition of flat netlists.

Full PDFs are available in the ``literatures/`` directory of the repository.
